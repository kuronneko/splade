## How to install [Docker]
```
clone project
cp .env.example .env
composer install

php artisan sail:install
0,3,5,7

./vendor/bin/sail up
./vendor/bin/sail php artisan key:generate
./vendor/bin/sail php artisan migrate
./vendor/bin/sail npm install
./vendor/bin/sail npm run dev
```
